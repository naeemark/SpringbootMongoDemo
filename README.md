# Spring boot RESTFul API using MongoDb 

A sample app using Java `Spring boot` and `MongoDB` for `sample` project

### Pre-reqs

- JDK 1.8
- IntelliJ

#### Functionality

- A `RESTful CRUD` mock api with response format `JSON`
- Provides functionality for `Hotel` reository
- **Calls**
  - **GET** 

#### Dependencies

- MongoDB
- Maven
- SpringBoot
